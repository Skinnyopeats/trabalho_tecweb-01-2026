# Agenda Web

Projeto desenvolvido para a disciplina de Tecnologias Web (IESB).  
Uma agenda simples para gerenciar compromissos e tarefas, com dados armazenados localmente no navegador.

## Tecnologias

- HTML5 semântico
- CSS3
- JavaScript (ES Modules)
- Vite
- LocalStorage

## Estrutura de páginas

| Página | Descrição |
|---|---|
| `index.html` | Home — próximos compromissos e itens recentes |
| `adicionar.html` | Formulário para cadastrar novos itens |
| `consultar.html` | Listagem completa com opção de exclusão |

## Como rodar localmente

```bash
cd myapp
npm install
npm run dev
```

Acesse `http://localhost:5173` no navegador.
